# DIY组装助手

一个面向DIY爱好者的电子产品维修与组装教程网站，涵盖983+类电子产品的零件清单与组装教程。

## 功能特点

- 🔍 **智能搜索与分类浏览**：支持按关键词搜索设备，20+设备分类
- 📋 **详细零件清单与比价**：每个设备都有完整的零件清单，包含价格和购买平台
- 📝 **维修步骤指南**：图文并茂的步骤说明，包含操作要点和注意事项
- 🤖 **AI智能助手**：自然语言问答，解答组装问题
- 💾 **本地存储**：聊天记录和收藏自动保存

## 技术栈

- HTML5 + CSS3 + JavaScript
- 纯前端实现，无需后端
- localStorage本地存储

## 快速开始

### 本地预览

直接用浏览器打开 `index.html` 即可。

或使用本地服务器：

```bash
# Python
python3 -m http.server 8000

# Node.js
npx http-server -p 8000
```

访问 `http://localhost:8000`

### GitHub Pages部署

1. 复制此仓库
2. 在GitHub仓库设置中开启Pages
3. 选择main分支，根目录
4. 访问 `https://<用户名>.github.io/<仓库名>`
